# Topdon TopScan Research

An open-source research project to understand, document, and extend the capabilities of the Topdon TopScan diagnostic interface.

## Objectives

- Document the hardware
- Identify all integrated circuits
- Map the OBD-II connector circuitry
- Analyze USB communications
- Investigate firmware architecture
- Determine J2534 compatibility
- Investigate ELM327 compatibility
- Investigate CAN, LIN, and SWCAN support
- Develop open-source software support

## Current Hardware

Main MCU:
- Nation N32G455

Current Status

- Device enumerates as a USB Serial Device on Windows.
- PCB has been disassembled for analysis.
- Hardware documentation is beginning.

## Planned Work

- Photograph PCB
- Identify every IC
- Trace OBD-II pin routing
- Capture USB traffic
- Analyze firmware
- Build open-source driver support
